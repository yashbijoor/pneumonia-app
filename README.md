## Deployment

### Unzip the models

### To deploy the server

```bash
  cd model
```
Install the packages
```bash
  pip install -r requirements.txt
```
Run the server
```bash
  flask run
```
### To deploy the interface

```bash
  cd app
```
Install the packages
```bash
  npm i
```
Run the react app
```bash
  npm start
```
